# История версий

Формат — [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
версионирование — [SemVer](https://semver.org/lang/ru/).
Текущая версия отображается на стартовом экране и задаётся константой
`VERSION` в `index.html`; при каждом релизе синхронно меняется имя кэша
в `sw.js` (`treska-vN`), чтобы PWA у игроков обновилась.

## [2.0.0] — 2026-06-11

Переработка ядра по итогам аудита производительности и трендов
([методология — в docs/OPTIMIZATION.md](docs/OPTIMIZATION.md)).

### Изменено
- **Рыбы переведены на GPU-инстансинг.** Каждый вид запекается в одну
  геометрию с цветом по вершинам и рендерится одним `InstancedMesh`:
  ~140 дроуколлов на рыб → **5**. Поведение, силуэты и виды не изменились.
- **Виляние хвостом перенесено в вершинный шейдер** (инстанс-атрибут
  `aPhase`): CPU больше не вращает кости/меши, только пишет матрицы.
- Шипы фугу теперь размещаются детерминированно («золотая спираль»)
  вместо случайных позиций — геометрия запекается один раз.

### Добавлено
- **Пространственный звук**: попадания, фанфары и взрывы панорамируются
  по направлению на источник и затухают с дистанцией (StereoPanner).
- **Процедурная реверберация**: ConvolverNode со сгенерированным импульсом —
  по-прежнему ноль аудиофайлов.
- **Адаптивное качество**: при просадке FPS плавно снижается масштаб
  рендера (до ×0.6) и возвращается при запасе; геймплей и UI не страдают.
- Дебаг-оверлей `?debug` — FPS, дроуколлы, треугольники, масштаб, рыбы.
- Номер версии на стартовом экране.
- Документация: `docs/ARCHITECTURE.md`, `docs/OPTIMIZATION.md`,
  `docs/GDD.md`, `docs/QA.md`, этот CHANGELOG.

### Исправлено
- Сведение снарядов: болты и дробь летят точно в перекрестье из любого
  положения орудия (целевая точка на 25 м / 12 м).
- Кэш service worker инвалидируется по версии (`treska-v2`).

## [1.0.0] — 2026-06-11

Первый релиз по ТЗ челленджа.

### Добавлено
- AR-режим: камера фоном, гироскоп 360°, iOS-разрешения по кнопке «СТАРТ».
- Фолбэки: «северное небо» с авророй и планктоном без камеры,
  драг-обзор без гироскопа, полная играбельность на десктопе.
- 5 видов процедурных рыб: треска, сайда («восьмёрки»), boids-стая мальков,
  золотая треска (слоумо при сбитии), фугу (−200, сброс комбо).
- 3 орудия: сдвоенная фотонная турель, рельсотрон (заряд → хитскан,
  мультикилл, воксельный распад), плазменный дробовик (веер 7 дробин).
- Полностью процедурный WebAudio-звук + адаптивный бит от комбо,
  «дзынь» комбо растёт на полтона за ступень, ветер и FM-чайки.
- Раунд 90 с, волны до 14 рыб, «золотая лихорадка», комбо ×2…×10,
  бонус за точность, прогрессия и скины в localStorage.
- HUD: счёт/рекорд/таймер/комбо-кольцо, радар-стрелки, чипы орудий.
- Супернаведение (магнетизм прицела, игнорирует фугу), Wake Lock,
  кнопка «📸 Поделиться» (Web Share / скачивание).
- PWA: manifest, service worker с офлайн-кэшем, netlify.toml,
  генератор PNG-иконок без зависимостей (`tools/make-icons.mjs`).
