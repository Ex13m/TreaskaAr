# 🐟 Подстрели треску — мобильный AR-тир

> Документация: [архитектура](docs/ARCHITECTURE.md) ·
> [оптимизация](docs/OPTIMIZATION.md) · [гейм-дизайн](docs/GDD.md) ·
> [QA-план](docs/QA.md) · [история версий](CHANGELOG.md)

Аркадный тир в дополненной реальности прямо в браузере телефона: камера — фон,
гироскоп крутит обзор на 360°, вокруг плавают процедурные 3D-рыбы, внизу —
футуристичное орудие. 90 секунд, комбо, рекорды, «ещё разок».

**Один файл `index.html`, vanilla JS + Three.js с CDN, ноль внешних ассетов:**
все рыбы собраны из примитивов, весь звук синтезирован WebAudio на лету.

---

## Запуск

Локально (камера/гироскоп требуют HTTPS либо localhost):

```bash
npx serve .        # или python3 -m http.server
```

Деплой: закинуть репозиторий на **Netlify** — `netlify.toml` уже настроен
(publish-корень, `no-cache` для `sw.js`, `Permissions-Policy` для камеры и
гироскопа). HTTPS обязателен, иначе браузер не выдаст ни камеру, ни сенсоры.

Иконки пересобираются командой `node tools/make-icons.mjs` (PNG пишется
вручную через zlib — без зависимостей).

### Свои звуки (стоковые сэмплы)

> **Из коробки уже подключены** 37 CC0-звуков Kenney и 2 атмосферных
> трека FreePD (см. [ASSETS.md](ASSETS.md)). Ниже — как заменить их своими.

По умолчанию весь звук синтезируется WebAudio (10 вариантов выстрелов,
FM-«металл», реверб, панорама). Если хочется студийных сэмплов — положи
лицензированные файлы в папку `sfx/` и опиши их в `sfx/manifest.json`:

```json
{
  "turret":  ["turret-1.mp3", "turret-2.mp3"],
  "rail":    ["rail-1.mp3"],
  "shotgun": ["shotgun-1.mp3", "shotgun-2.mp3"],
  "hit":     ["hit-1.mp3", "hit-2.mp3", "hit-3.mp3"],
  "trophy":  ["trophy.mp3"]
}
```

Игра подхватит их автоматически (round-robin + случайный питч ±6% +
пространственная панорама); без папки молча останется на синтезе.
Хорошие источники CC0: kenney.nl (Sci-Fi Sounds), freesound.org,
sonniss.com GDC bundles — проверь лицензию перед коммитом файлов в репо.

### Своя музыка

По умолчанию играет генеративный арктический эмбиент (пэды Am–F–C–Em с
реверберацией). Чтобы заменить его на трек — положи лицензированный файл в
`music/` и создай `music/manifest.json`:

```json
{ "tracks": ["arctic-theme.mp3", "alt-theme.mp3"] }
```

Игра выберет случайный трек и зациклит на громкости ~22%. Источники
CC0-музыки: opengameart.org (раздел CC0 Music), itch.io (теги cc0+music),
Pixabay Music, Free Music Archive — лицензию проверяй перед коммитом.

## Анализ ТЗ: ключевые риски и принятые решения

| Риск из ТЗ | Решение |
|---|---|
| iOS отдаёт гироскоп только из жеста | `DeviceOrientationEvent.requestPermission()` вызывается строго внутри клика «▶ СТАРТ», там же `getUserMedia` и резюм AudioContext — один жест, три разрешения |
| Нет камеры / отказ | Шейдерное «северное небо»: градиент + процедурная аврора + звёзды + дрейфующий планктон (Points) |
| Нет гироскопа / десктоп | Обзор перетаскиванием (pointer events), полный геймплей мышью, клавиши 1/2/3 — смена орудий |
| Конфликт «тап = выстрел» и «драг = обзор» | Порог 12 px: до него — выстрел/заряд, после — обзор; начатый заряд рельсотрона при драге тихо отменяется |
| 60 FPS на среднем телефоне | Пулы всего: снаряды (32), частицы (10×36), кольца, воксели, всплывающие цифры; общие геометрии/материалы у рыб; `pixelRatio ≤ 2`; ноль теней; один PointLight на вспышки |
| Офлайн | SW кэширует ядро + CDN-модуль Three.js; офлайн открывается стартовый экран и честно сообщает про камеру |

## Что в игре (по ТЗ)

- **5 видов рыб** из примитивов: треска (усик, три спинных плавника), сайда
  «восьмёрками», boids-стая мальков, золотая треска с emissive-пульсацией,
  ореолом и фанфарой (7% спавнов, ×3 в «золотую лихорадку»), фугу с шипами
  и красным миганием (−200, встряска, сброс комбо).
- **3 орудия с разным характером**: фотонная турель (очередь 160 мс,
  попеременные стволы, откат + вспышка), рельсотрон (заряд с воем и
  светящимися конденсаторами → хитскан-луч сквозь всех на линии, мультикилл,
  кик камеры, распад рыбы на воксели, перезарядка с «паром»), плазменный
  дробовик (веер 7 дробин ~12°, «прожарка» и жирный брызг).
- **Четырёхканальный фидбек**: частицы + ударное кольцо, радиальная вспышка и
  scale-pop прицела, `navigator.vibrate` (35 мс / [30,40,60] / длинная),
  всплывающие «+150» из 3D-точки. Промах гаснет искрой и пугает рыбу рядом.
- **Звук — 100% WebAudio-синтез**: pew 800→110 Гц, вой заряда 80→600 Гц,
  шумовой «бах» тремя слоями, арпеджио трофея, диссонанс фугу, «дзынь» комбо
  растёт на полтона за ступень, ветер + FM-чайки, мастер-лимитер, mute в HUD.
- **Цикл**: 90 с, волны до 14 рыб, финальная «золотая лихорадка» с красным
  пульсом таймера, комбо ×2…×10 (сброс только по таймеру 2.5 с, на ×5/×10 —
  синт-стингер), бонус +10% за точность >60%, прогрессия в localStorage
  (рельсотрон 2000, дробовик 5000, скины 10000: неон/хром/ржавый панк).
- **HUD**: счёт/рекорд, таймер, комбо-кольцо, прицел с «дыханием»,
  чипы орудий с барами заряда/перезарядки, радар-стрелки на рыбу за кадром
  (золотая — в приоритете), тёмный неон + glassmorphism + шрифт Unbounded.
- **PWA**: manifest (fullscreen), service worker, установка на домашний экран.

## Сверх ТЗ — чтобы «залетело» ✨

Реализовано уже сейчас:

1. **Golden Moment** — слоумо ×0.25 на полсекунды при сбитии золотой трески:
   конфетти зависает в воздухе, кадр сам просится в сторис.
2. **Адаптивный саундтрек** — бит просыпается вместе с комбо: на ×2 появляется
   кик, на ×3 хэты, на ×4 бас, на ×5 пентатонический арп. Комбо реально
   *слышно*, музыка собирается из игры, а не играет фоном.
3. **📸 Поделиться** — кнопка на экране результатов склеивает кадр камеры,
   3D-сцену и плашку счёта в картинку и шарит через Web Share API
   (фолбэк — скачивание). Готовый вирусный контент в один тап.
4. **Супернаведение** — мягкий магнетизм прицела (~3°, для рельсотрона 1.5°):
   игра прощает дрожь рук на гироскопе, но фугу магнит честно игнорирует —
   в бомбу можно попасть только самому.
5. **Аврора** — в фолбэке без камеры небо не заглушка, а процедурное северное
   сияние на шейдере: зелёно-фиолетовые ленты, звёзды, планктон.
6. **Wake Lock** — экран не гаснет посреди раунда.

## Roadmap: предложения следующих итераций

- **Сетевая игра** (по нарастанию сложности):
  1. *Глобальный лидерборд* — Netlify Function + KV/Upstash, топ дня и топ
     недели прямо на экране результатов (день работы).
  2. *Ghost-дуэль* — асинхронный PvP: после раунда выгружается «запись»
     (тайминги выстрелов/фрагов), друг по ссылке играет тот же сид спавна и
     видит призрачные фраги соперника в реальном времени. Без серверной
     логики — статика + один JSON.
  3. *Live-дуэль по WebRTC* — два игрока, общий сид, сплит счёта в HUD,
     сигналинг через лёгкую функцию; «украденный» трофей соперника — главная
     эмоция режима.
- **Клипы для TikTok/Reels** — MediaRecorder пишет последние 15 секунд канваса
  (камера + 3D + HUD), после рекорда предлагает сохранить вертикальное видео.
- **Ежедневный челлендж** — сид спавна от даты: у всех игроков одинаковая
  «раздача» рыб, честное сравнение результатов, причина возвращаться каждый день.
- **Боссы** — кит-левиафан раз в N раундов: общая полоска HP, фазы, уязвимые точки.
- **Графика+** — WebGPU-рендерер с фолбэком на WebGL, bloom-постпроцессинг на
  трофеях и луче рельсотрона, каустика света на рыбах.
- **WebXR** — режим `immersive-ar` с плоскостями (рыбы прячутся за мебелью)
  там, где доступен, с фолбэком на текущий гиро-режим.
