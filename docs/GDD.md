# Гейм-дизайн-документ (GDD)

## Фантазия и эмоция

Игрок — стрелок в центре арктического неба, полного парящей рыбы.
Сессия 90 секунд, главная эмоция — **«ещё разок»**: короткая партия,
видимый прогресс (рекорд/разблокировки), момент славы (золотая треска
в слоумо), которым хочется поделиться.

##核 (core loop), 30 секунд

```
осмотрись (гироскоп/радар) → выбери цель → стреляй → комбо растёт →
слышишь, как музыка собирается из твоего комбо → не сорвись на фугу →
волна плотнее → золотая! → слоумо → скриншот → «ЕЩЁ РАЗ»
```

## Бестиарий и баланс

| Вид | Очки | Скорость | Поведение | Роль в балансе |
|---|---|---|---|---|
| Треска | 100 | медленная | плавный steering | «хлеб», учит целиться |
| Сайда | 150 | быстрая | восьмёрки Лиссажу | тест упреждения |
| Мальки ×5–7 | 50/шт | средняя | boids-стая | пища для дробовика и комбо |
| Золотая треска | 500 | медленная | как треска, ореол + фанфара | джекпот, 7% (×3 в финале) |
| Фугу | −200 | медленная | мигает красным | анти-спам: сбрасывает комбо |

Распределение спавна: золотая 7% → фугу 12% → стая 15% → сайда 25% → треска
остальное. В «золотую лихорадку» (последние 10 с) шанс золотой ×3.

## Орудия: правило ниш (16 стволов)

Каждое орудие выигрывает в своей ситуации и проигрывает в чужой —
переключение чипов и есть скилл-экспрессия. Лестница разблокировок —
по сумме очков за все игры.

| Орудие | Анлок | Режим | Ниша |
|---|---|---|---|
| ⚡ Турель | 0 | авто, 160 мс | одиночки на любой дистанции |
| 🔴 Бластер | 1000 | авто, спарка | быстрые цели, высокая скорость болта |
| 🔱 Рельсотрон | 2000 | заряд→хитскан | линия рыб, мультикилл |
| 🪓 Топор | 3500 | тап, дуга+кувырок | навесом за укрытие радара, пробивает двоих |
| 💥 Дробовик | 5000 | тап, веер 7×12° | стая в упор |
| 🟢 Плазмомёт | 6500 | тап, AoE-шар | кучные группы на средней |
| 🌀 Вулкан | 8000 | авто после раскрутки | подавление, «золотая лихорадка» |
| 🌩 Тесла | 9000 | тап, цепь ×3 | гарантированное комбо без прицеливания |
| 🎆 Флак | 11000 | тап, прокси-взрыв | быстрые сайды, воздушный подрыв |
| 🚀 Рой | 12000 | тап, 3 самонавод. | «выстрелил-забыл», дальние цели |
| 💿 Диск | 14000 | тап, рикошет ×3 | растянутые группы |
| ❄️ Крио-луч | 15000 | луч | контроль: вязнут и раскалываются |
| 📌 Игломёт | 16000 | авто, лёгкий самонавод | неаккуратная стрельба прощается |
| 🎯 Гаусс | 18000 | тап, хитскан | снайпер: очки ×1.5 |
| ☢️ БФГ-мини | 20000 | заряд, плавящий шар | зачистка экрана |
| 🕳️ Сингулярность | 25000 | тап | затягивает стаю и схлопывает |

Критерий из ТЗ сохранён: орудия различимы **с закрытыми глазами** —
у каждого свой синтез-рецепт и вибро-подпись. Опасность фугу осталась:
AoE-оружие взрывает и её (−200) — площадь это не только сила.

## Комбо и экономика

- Окно 2.5 с, попадание наращивает множитель до ×10.
  **Промах не сбрасывает** (сброс только по таймеру или фугу) — агрессия
  поощряется, паника нет.
- Слышимость прогресса: «дзынь» каждой ступени выше на полтона; на ×2
  просыпается кик, ×3 хэты, ×4 бас, ×5 арп (см. `musicTick`).
- Очки = база × множитель; финальный бонус +10% за точность > 60%.
- Прогрессия на **сумме очков за все игры** (не за рекорд): даже слабая
  партия двигает к рельсотрону (2000), дробовику (5000), скинам (10000).
  Хранится в localStorage — без аккаунтов и трения.

## Кривая раунда (90 с)

| Время | Рыб одновременно | Событие |
|---|---|---|
| 0–15 с | 4 | разогрев, только что заспавнились |
| каждые 15 с | +2 | волна |
| 75–80 с | 12–14 | пик плотности |
| 80–90 с | 12–14 | «золотая лихорадка»: таймер пульсирует, золотые ×3 |

## Фидбек попадания — 4 канала одновременно

1. **3D**: взрыв частиц цвета орудия + ударное кольцо (+ воксели у рельсы,
   конфетти у золотой);
2. **экран**: радиальная вспышка, scale-pop прицела, shake на мультикилле;
3. **хаптика**: 35 мс / [30,40,60] трофей / длинная фугу;
4. **цифры**: «+150» из точки попадания, крупнее и фиолетовым при комбо.

Промах тоже отыгрывается: искра на излёте + испуг рыбы рядом (рыба
дёргается — игрок видит, что был близко).

## Доступность и честность

- Супернаведение ~3° прощает дрожь гироскопа, но фугу не притягивает.
- Полный фолбэк-стек: без камеры, без гироскопа, на десктопе — игра
  обязана быть играбельной, а не «извиниться и закрыться».
- Разрешения запрашиваются только по явному действию (кнопка СТАРТ),
  с объяснением зачем.
