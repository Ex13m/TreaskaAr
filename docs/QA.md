# QA: план тестирования

## Матрица устройств (минимум)

| Среда | Что проверяем |
|---|---|
| iPhone, iOS Safari | requestPermission гироскопа из жеста, камера, вибро (нет — ок), PWA A2HS |
| Android среднего класса, Chrome | 60 FPS под нагрузкой, вибрация, установка PWA |
| Android слабый / throttling ×4 | адаптивное качество снижает масштаб, играбельность |
| Десктоп Chrome/Firefox | мышь: драг-обзор, клик, удержание; клавиши 1/2/3 |
| Десктоп без камеры | фолбэк «северное небо» |
| Офлайн (после первого визита) | стартовый экран открывается, честное сообщение про камеру |

## Чеклист релиза

### Разрешения и старт
- [ ] До нажатия СТАРТ — ни одного запроса разрешений.
- [ ] Отказ от камеры → аврора-фолбэк + тост, игра идёт.
- [ ] Отказ от гироскопа (iOS) → драг-обзор работает.
- [ ] Повторный СТАРТ не создаёт второй AudioContext/поток камеры.

### Геймплей
- [ ] Тап стреляет; смещение > 12 px — обзор без выстрела.
- [ ] Турель: очередь при удержании, стволы чередуются.
- [ ] Рельсотрон: заряд < 0.3 с — холостой; драг отменяет заряд и звук;
      луч пробивает несколько рыб (мультикилл-тост).
- [ ] Дробовик: 7 дробин, у стаи мальков ловит 2+ за залп.
- [ ] Фугу: −200, сброс комбо, красная вспышка, длинная вибрация.
- [ ] Золотая: фанфара при споуне, приоритетная стрелка радара, слоумо при сбитии.
- [ ] Комбо не сбрасывается промахом, сбрасывается по таймеру 2.5 с.
- [ ] Последние 10 с: таймер пульсирует, золотые заметно чаще.
- [ ] Конец раунда: точность ≤ 100%, бонус начисляется при > 60%.
- [ ] Рекорд: фейерверк + плашка; «ЕЩЁ РАЗ» мгновенно перезапускает.

### Прогрессия
- [ ] 2000/5000 суммарных очков открывают чипы прямо во время игры (тост).
- [ ] Скины применяются и переживают перезагрузку (localStorage).
- [ ] Локальные данные не теряются при обновлении версии.

### Производительность (на телефоне среднего класса, `?debug`)
- [ ] «Золотая лихорадка» + дробовик + 3 взрыва: fps ≥ 55, calls < 40.
- [ ] JS heap без «пилы» в Performance Monitor (нет аллокаций в кадре).
- [ ] 3 минуты игры — без роста памяти и деградации FPS.
- [ ] Принудительный лаг (throttling) → масштаб рендера падает до 0.6 и
      возвращается к 1.0 после снятия нагрузки.

### Звук
- [ ] Три орудия различимы вслепую (звук + вибрация).
- [ ] Попадание слева слышно слева (панорама).
- [ ] Mute мгновенный и полный (включая реверб-хвост), переживает релоад.
- [ ] Сворачивание вкладки глушит звук, возврат — восстанавливает.
- [ ] Комбо-бит стихает при сбросе комбо, не играет в меню.

### PWA / инфраструктура
- [ ] Lighthouse: PWA installable, без ошибок консоли во всех режимах.
- [ ] `node --check` на извлечённом `<script type="module">`, `sw.js`,
      `tools/make-icons.mjs` — чисто.
- [ ] Смена `VERSION` + `CACHE` → у установленного PWA новая версия со
      второго запуска.
- [ ] Заголовки Netlify: `sw.js` no-cache, Permissions-Policy отдаётся.

## Регресс-якоря (что чаще всего ломается)

1. Порядок «жест → разрешения → AudioContext.resume» на iOS.
2. Конфликт tap/drag после смены обработчиков ввода.
3. Невидимые рыбы после смены `max` у вида (free-list рассинхрон).
4. Шейдерный инжект после апгрейда Three.js (поменялись чанки) —
   проверять, что хвосты виляют.
